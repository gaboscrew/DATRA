# Datra Landing

## Instalar
```bash
npm install
```

## Correr en local
```bash
npm run dev
```

## Subir a Vercel
1. Sube esta carpeta a GitHub o arrástrala en Vercel.
2. Vercel detecta Next.js automáticamente.
3. Haz deploy.

## Dominio
Después puedes conectar tu dominio desde el panel de Vercel.
